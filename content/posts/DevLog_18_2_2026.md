# The objetive of the day
Our goal for tthis wednesday was to understand two things
1. what is Json?
2. what is a RESTful API? 

The very short explanation for these two questions is:
1. Json or JavaScript Object Notation is a file format we use to transport data between aplications
2. A RESTful API is a way we request data between aplications though HTTP methods.

We had to show our understanding of these elements by creating a new aplication that got data through a RESTful API from TMdb.

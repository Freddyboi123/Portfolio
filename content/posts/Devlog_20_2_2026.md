# API and Threads
Today we had to complete an assignment where we had to request information from mulitple api's, and convert serialize the information into Java objects. 
after we got this to work, we had to implimment a version that fetched all the information form the API's using a single thread, and one that used three threads to get the inormation independetnly.

from my solution we can see that using a single thread to solve this problem takes around 3000msm and using three threads to solve the same problem, only takes around 400ms. 
this tells us that solving the problem with the right tools can save up to 9 times the time in this specific case.
